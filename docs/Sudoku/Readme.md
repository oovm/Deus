# Functions

Sudoku 模块

::: tip 数独列表式
约定 n 阶数独是一个 $n^2×n^2$ 的矩阵
正数表示原本就存在的值, 也就是谜题
0 表示未知的值, 需要求解的部分
负数表示用户填入的部分, 或者求解函数返回的部分
其他非整数非数字都按填 0 处理
:::

::: tip 数独字符串
约定 n 阶数独是一个 $n^4$ 长的字符串
0 表示未知的值, 需要求解的部分
然后从 1 到 9, a-z , A-Z 依次编码
如果还不够的话还能用希腊字母, 其他 各种字母填充
大型数独不建议用这种方式编码
:::
